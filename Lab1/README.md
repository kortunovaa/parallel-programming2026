# Лабораторная работа №1
### Умножение квадратных матриц

## Задание

Написать программу на языке C/C++ для перемножения двух квадратных матриц.

**Требования:**
- **Исходные данные:** файлы, содержащие значения исходных матриц
- **Выходные данные:** файлы со значениями результирующей матрицы, время выполнения, объем задачи
- **Обязательна автоматизированная верификация** результатов вычислений с помощью сторонних библиотек (Python/NumPy)

## Реализация

Программа реализует классический алгоритм умножения матриц с тремя вложенными циклами. Этот алгоритм является прямым следствием математического определения произведения матриц.
**Математическая основа**
Произведение двух матриц A и B размера N×N определяется как:
C[i][j] = Σ(k=1 to N) A[i][k] * B[k][j], где i, j, k принимают значения от 0 до N-1.

Временная сложность: O(N³)
Объем вычислений: 2 * N³ операций, где:
N³ операций умножения
N³ операций сложения
