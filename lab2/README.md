## Задание

Модифицировать программу из лабораторной работы №1 для параллельного выполнения с использованием технологии OpenMP.

**Требования:**
Провести серию экспериментов с разным количеством потоков (1, 2, 4, 8)
Провести эксперименты с разными размерами матриц (200, 400, 800, 1200, 1600, 2000)
Исследовать влияние количества вычислительных ядер на производительность
Выполнить автоматизированную верификацию результатов

## Реализация

В программу из лабораторной работы №1 были добавлены директивы OpenMP для распараллеливания вычислений:
```cpp
vector<vector<double>> Multiply_Matrix(const vector<vector<double>>& A,
    const vector<vector<double>>& B,
    int num_threads) {

    int n = A.size();
    vector<vector<double>> result(n, vector<double>(n, 0));

    omp_set_num_threads(num_threads);  // установка количества потоков

#pragma omp parallel for schedule(dynamic)  // распараллеливание цикла
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            for (int k = 0; k < n; k++) {
                result[i][j] += A[i][k] * B[k][j];
            }
        }
    }
    return result;
}
```


## Методика проведения экспериментов
1)Генерация матриц: Python-скрипт generate_matrix.py создает матрицы размеров 200, 400, 800, 1200, 1600, 2000
2)Количество потоков: 1, 2, 4, 8
3)Количество повторений: 3 замера для каждой комбинации (размер × потоки)
4)Автоматизация: test.py запускает все эксперименты последовательно
5)Верификация: paral.py проверяет корректность результатов с помощью NumPy
